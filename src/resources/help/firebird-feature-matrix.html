Firebird Feature Matrix

This page provides an overview of important features introduced in different Firebird versions.

Version Main Features
1.5 (2003) • Stored Procedures & Triggers
• User Defined Functions (UDF)
• Generators (Sequences)
• Basic SQL functions (SUBSTRING, CAST, UPPER, …)
2.0 / 2.1 (2006–2008) • EXECUTE BLOCK (anonymous procedures)
• Common Table Expressions (CTE, WITH …)
• Extended Monitoring Tables (MON$…)
• MERGE statement
2.5 (2010) • Multithreaded engine (SMP support)
• Superserver connection pooling
• Enhanced monitoring (deadlocks, I/O stats)
• Trace & Audit services
• No native SQL functions (only Stored Procedures & UDFs)
3.0 (2016) • Native SQL functions (CREATE FUNCTION)
• Packages (grouping of procedures & functions)
• Global Temporary Tables (GTT)
• Roles with admin rights
• New authentication plugins (SRP, Legacy_Auth, …)
• DDL allowed inside PSQL
4.0 (2021) • BOOLEAN data type
• INT128 and DecFloat
• TIMESTAMP WITH TIME ZONE
• Read Consistency (improved MVCC)
• Lock timeouts & wait options
• Scrollable cursors in PSQL
5.0 (2023) • Parallel execution (e.g. SELECT COUNT(*))
• Extended Window Functions
• CREATE OR ALTER for procedures & functions
• Optimizer and monitoring improvements
6.0 (planned) • More parallelization (joins, aggregates)
• Additional ANSI SQL features
• Further optimizer improvements
• Performance tuning
